# Komandara Wizard

A VSCode extension that scaffolds Komandara RISC-V projects.

## Features

- Scaffolds a complete Komandara C software project.
- Downloads `k10.h` and `startup.S` for immediate compilation.
- Provisions `launch.json` and OpenOCD config tailored to the selected development board (e.g. Genesys 2, Simulation).

## Requirements

- RISC-V Cross Compiler Toolchain
- OpenOCD installed for hardware debugging.

## Usage

Hit `Ctrl+Shift+P` and execute **Komandara: Create Project**.
